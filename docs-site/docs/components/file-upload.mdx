## With Restrictions

```jsx
<FileUpload
  accept="image/*,.pdf"
  multiple
  maxFiles={5}
  maxFileSize={10 * 1024 * 1024}
  onFilesChange={(files) => console.log(files)}
  onValidationError={(errors) => console.error(errors)}
/>
```
