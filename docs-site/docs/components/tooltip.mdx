# Tooltip

A popup that displays informative text when hovering or focusing on a trigger element.

## Basic Usage

<ComponentPreview code={`<Tooltip content="Hello from tooltip!">
  <Button>Hover me</Button>
</Tooltip>`}>
  <Tooltip content="Hello from tooltip!">
    <Button>Hover me</Button>
  </Tooltip>
</ComponentPreview>
