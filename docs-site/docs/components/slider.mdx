# Slider

A range input for selecting a numeric value within a given range.

## Basic Usage

<ComponentPreview code={`<Slider defaultValue={50} />`}>
  <div style={{ width: '100%', maxWidth: 320 }}>
    <Slider defaultValue={50} />
  </div>
</ComponentPreview>

## With Value Label

<ComponentPreview code={`<Slider defaultValue={40} showValue />`}>
  <div style={{ width: '100%', maxWidth: 320 }}>
    <Slider defaultValue={40} showValue />
  </div>
</ComponentPreview>
