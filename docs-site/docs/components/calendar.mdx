# Calendar

A standalone calendar grid used internally by DatePicker and DateRangePicker. Can also be used independently to display a month view.

## Basic Usage

<ComponentPreview code={`<Calendar
  currentMonth={2}
  currentYear={2026}
  onDateClick={(date) => console.log(date)}
/>`}>
  <Calendar
    currentMonth={2}
    currentYear={2026}
    onDateClick={(date) => {}}
  />
</ComponentPreview>
