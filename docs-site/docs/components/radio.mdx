# Radio

A set of mutually exclusive options. Use `RadioGroup` to wrap multiple `Radio` components.

## Basic Usage

<ComponentPreview code={`<RadioGroup name="fruit" defaultValue="apple">
  <Radio value="apple" label="Apple" />
  <Radio value="banana" label="Banana" />
  <Radio value="cherry" label="Cherry" />
</RadioGroup>`}>
  <RadioGroup name="fruit" defaultValue="apple">
    <Radio value="apple" label="Apple" />
    <Radio value="banana" label="Banana" />
    <Radio value="cherry" label="Cherry" />
  </RadioGroup>
</ComponentPreview>

## Horizontal Layout

<ComponentPreview code={`<RadioGroup name="size" defaultValue="md" direction="horizontal">
  <Radio value="sm" label="Small" />
  <Radio value="md" label="Medium" />
  <Radio value="lg" label="Large" />
</RadioGroup>`}>
  <RadioGroup name="size" defaultValue="md" direction="horizontal">
    <Radio value="sm" label="Small" />
    <Radio value="md" label="Medium" />
    <Radio value="lg" label="Large" />
  </RadioGroup>
</ComponentPreview>
