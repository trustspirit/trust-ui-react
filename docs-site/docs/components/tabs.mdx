# Tabs

Organizes content into tabbed sections. Supports underline and pill variants with keyboard navigation.

    <Tabs.Trigger value="tab3">Notifications</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Content value="tab1">Account settings content</Tabs.Content>
  <Tabs.Content value="tab2">Security settings content</Tabs.Content>
  <Tabs.Content value="tab3">Notification preferences content</Tabs.Content>
</Tabs>`}>
