## Fill style

`fillStyle` separates the **color intent** (variant) from the **visual weight** (fill). Pick `subtle` for low-emphasis status labels, `solid` for high-contrast emphasis, `outline` when the badge sits on a busy surface where solid would clash.

<ComponentPreview code={`<Badge variant="success" fillStyle="solid">Solid</Badge>
<Badge variant="success" fillStyle="subtle">Subtle</Badge>
<Badge variant="success" fillStyle="outline">Outline</Badge>`}>
  <Badge variant="success" fillStyle="solid">Solid</Badge>
  <Badge variant="success" fillStyle="subtle">Subtle</Badge>
  <Badge variant="success" fillStyle="outline">Outline</Badge>
</ComponentPreview>
