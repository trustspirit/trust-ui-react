## Variant (fill style)

`variant` controls the visual weight — pick `subtle` for low-emphasis status labels, `solid` for high-contrast emphasis, `outline` when the badge sits on a busy surface where solid would clash.

<ComponentPreview code={`<Badge color="success" variant="solid">Solid</Badge>
<Badge color="success" variant="subtle">Subtle</Badge>
<Badge color="success" variant="outline">Outline</Badge>`}>

  <Badge color="success" variant="subtle">Subtle</Badge>
  <Badge color="success" variant="outline">Outline</Badge>
</ComponentPreview>
