# Avatar

Displays a user image, initials fallback, or a default icon.

</ComponentPreview>

## Status indicator

Pass `status` to render a colored dot in the bottom-right corner — useful for presence in chat / collaboration UI.

<ComponentPreview code={`<Avatar name="Alice" status="online" />
<Avatar name="Bob" status="away" />
<Avatar name="Charlie" status="busy" />
<Avatar name="Dana" status="offline" />`}>
  <Avatar name="Alice" status="online" />
  <Avatar name="Bob" status="away" />
  <Avatar name="Charlie" status="busy" />
  <Avatar name="Dana" status="offline" />
</ComponentPreview>

## Outlined (stacked)

For overlapping avatar stacks, set `outlined` to add a ring matching the page background — separates each avatar from its neighbor without needing manual gaps.

<ComponentPreview code={`<Avatar name="A" outlined />
<Avatar name="B" outlined style={{ marginLeft: -10 }} />
<Avatar name="C" outlined style={{ marginLeft: -10 }} />`}>
  <Avatar name="A" outlined />
  <Avatar name="B" outlined style={{ marginLeft: -10 }} />
  <Avatar name="C" outlined style={{ marginLeft: -10 }} />
</ComponentPreview>

## Default Fallback

When no `src` or `name` is provided, a generic user icon is shown.

<ComponentPreview code={`<Avatar size="sm" />
<Avatar size="md" />
<Avatar size="lg" />`}>
  <Avatar size="sm" />
  <Avatar size="md" />
  <Avatar size="lg" />
</ComponentPreview>
