# Avatar

Displays a user image, initials fallback, or a default icon.

## With Initials

<ComponentPreview code={`<Avatar name="John Doe" size="sm" />
<Avatar name="John Doe" size="md" />
<Avatar name="John Doe" size="lg" />`}>
  <Avatar name="John Doe" size="sm" />
  <Avatar name="John Doe" size="md" />
  <Avatar name="John Doe" size="lg" />
</ComponentPreview>

## Shapes

<ComponentPreview code={`<Avatar name="Alice" shape="circle" />
<Avatar name="Alice" shape="square" />`}>
  <Avatar name="Alice" shape="circle" />
  <Avatar name="Alice" shape="square" />
</ComponentPreview>

## Default Fallback

When no `src` or `name` is provided, a generic user icon is shown.

<ComponentPreview code={`<Avatar size="sm" />
<Avatar size="md" />
<Avatar size="lg" />`}>
  <Avatar size="sm" />
  <Avatar size="md" />
  <Avatar size="lg" />
</ComponentPreview>
