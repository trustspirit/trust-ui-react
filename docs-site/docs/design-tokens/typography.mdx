# Typography

## Font Family

```css
--tui-font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
```

## Font Sizes

| Token | Value |
|-------|-------|
| `--tui-font-size-xs` | 0.75rem (12px) |
| `--tui-font-size-sm` | 0.875rem (14px) |
| `--tui-font-size-md` | 1rem (16px) |
| `--tui-font-size-lg` | 1.125rem (18px) |
| `--tui-font-size-xl` | 1.25rem (20px) |

## Font Weights

| Token | Value |
|-------|-------|
| `--tui-font-weight-normal` | 400 |
| `--tui-font-weight-medium` | 500 |
| `--tui-font-weight-semibold` | 600 |
| `--tui-font-weight-bold` | 700 |

## Line Height

| Token | Value |
|-------|-------|
| `--tui-line-height` | 1.5 |

## Transitions

| Token | Value | Usage |
|-------|-------|-------|
| `--tui-transition-fast` | 100ms ease | Micro-interactions |
| `--tui-transition` | 150ms ease | Default transitions |
| `--tui-transition-slow` | 300ms ease | Larger animations |
