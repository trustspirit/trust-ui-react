# Colors

trust-ui uses CSS Custom Properties for all colors, supporting light and dark themes.

## Primary

| Token | Usage |
|-------|-------|
| `--tui-primary` | Primary actions, links |
| `--tui-primary-hover` | Primary hover state |
| `--tui-primary-active` | Primary active state |
| `--tui-primary-subtle` | Primary backgrounds |
| `--tui-primary-text` | Text on primary bg |

## Semantic Colors

| Token | Usage |
|-------|-------|
| `--tui-success` | Success states |
| `--tui-success-hover` | Success hover |
| `--tui-success-subtle` | Success background |
| `--tui-danger` | Error/danger states |
| `--tui-danger-hover` | Danger hover |
| `--tui-danger-subtle` | Danger background |
| `--tui-warning` | Warning states |
| `--tui-warning-hover` | Warning hover |
| `--tui-warning-subtle` | Warning background |
| `--tui-info` | Informational states |
| `--tui-info-hover` | Info hover |
| `--tui-info-subtle` | Info background |

## Surface & Text

| Token | Usage |
|-------|-------|
| `--tui-bg` | Page background |
| `--tui-bg-subtle` | Subtle background |
| `--tui-bg-muted` | Muted background |
| `--tui-bg-hover` | Hover background |
| `--tui-text` | Primary text |
| `--tui-text-secondary` | Secondary text |
| `--tui-text-muted` | Muted text |
| `--tui-text-inverse` | Inverse text |

## Border

| Token | Usage |
|-------|-------|
| `--tui-border` | Default borders |
| `--tui-border-hover` | Hover borders |
| `--tui-border-focus` | Focus borders |

## Other

| Token | Usage |
|-------|-------|
| `--tui-focus-ring` | Focus ring color |
| `--tui-overlay` | Modal overlay |
